Clase 5: HTML y sus Estilos

Fundamentos e Introducción


Definición:

HTML (HyperText Markup Language) es un lenguaje de programación que se utiliza para crear páginas web. Es el lenguaje estándar para estructurar y mostrar el contenido de una página web.

¿Qué hace HTML?

  • Define la jerarquía de títulos
  • Organiza los párrafos
  • Define la ubicación de las imágenes
  • Permite crear secciones, párrafos y enlaces
  • Define cómo se va a presentar el contenido en un navegador web

Historia de HTML

La primera versión de HTML fue escrita por Tim Berners-Lee en 1991. La versión más reciente es HTML 5.0, lanzada en 2006. La cual es la versión más utilizada en la actualidad.

Ventajas de HTML

  • Es gratuito y de código abierto
  • Se ejecuta en todos los navegadores web
  • Se integra fácilmente con otros lenguajes, como PHP y Node.js

Desventajas de HTML

  • Se utiliza principalmente para páginas web estáticas
  • Para una funcionalidad dinámica, puede ser necesario utilizar JavaScript o un lenguaje de back-end como PHP
